# Seatwise Agent Rules

## Always Use Skills
When working on this project, ALWAYS check and use the available skills in `.agent/skills/`.

### For Any UI/Frontend Work
First read: `@frontend-design`, `@ui-ux-pro-max`

### For Any Excel/Data Work  
First read: `@xlsx`

### For Any PDF Work
First read: `@pdf`

### For Any Testing
First read: `@webapp-testing`, `@systematic-debugging`

### For Complex Tasks
First read: `@brainstorming`, `@writing-plans`

## Project Context
- Static HTML/CSS/JS web app
- Uses CDN libraries (jsPDF, SheetJS, RemixIcon)
- No npm/package.json - all deps via CDN
- Run with Live Server or http-server
